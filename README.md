# Pin

Перезапуск ипереосмысление сервиса доставки [Raketa](https://github.com/Open-Rocket/raketa_delivery.git)

## Основная мысль

Pin — это open-source сервис городских задач на карте. Он создаёт временный слой намерений поверх города и позволяет людям находить друг друга для решения простых задач рядом. Пользователь ставит Pin в нужном месте, описывает, что нужно сделать, указывает цену и контакт для связи. Pin живёт ограниченное время и виден людям поблизости. В Pin нет ролей заказчика и исполнителя — каждый пользователь может быть и тем, и другим. Проект строится вокруг простоты, прямого контакта между людьми и отказа от лишних посредников, алгоритмов и комиссий. Pin не является доставкой, такси или маркетплейсом услуг — это базовая инфраструктура для взаимодействия людей в городе.
